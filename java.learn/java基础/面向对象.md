### 类和对象概述：

Class & Object

类泛指一大类抽象事物，对象特指这类事物中单一个体

类：属性，方法



#### 设计类

```java
public class Phone{
    //属性
    String name;
    double price;
    //方法
    public void call(){
        System.out.println("打电话");
    }
    
}
```



#### 创建对象

```java
Phone p1 = new Phone;
p1.name = "华为";
p1.call();
```

### 类的构造器

作用1：搭配new关键字，创建类的对象

作用2：在创建对象时，给相关属性赋值

没有显示提供构造器时，会默认生成一个空参的构造器。权限和类的权限相同

一个类中可以有多个构造器，他们之间构成重载





### 属性赋值过程



1. 默认赋值

   即给出的默认值

2. 显式赋值

   在类中定义时赋值

3. 构造器赋值

4. 通过  对象.方法  赋值

5. 通过  对象.属性  赋值



先后顺序：

1 - 2 - 3 - 4 / 5
