## 面向对象编程   Object

### 类和对象概述：

Class & Object

类泛指一大类抽象事物，对象特指这类事物中单一个体

类：属性，方法



#### 设计类

```java
public class Phone{
    //属性
    String name;
    double price;
    //方法
    public void call(){
        System.out.println("打电话");
    }
    
}
```



#### 创建对象

```java
Phone p1 = new Phone;
p1.name = "华为";
p1.call();
```
